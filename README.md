# 🧾 POS System (Full Stack Template)

A modern, scalable **Point of Sale (POS) system template** built with latest web technologies.  
This project is designed as a **reusable starter template** for future POS or admin dashboard applications.

---

# 🚀 Tech Stack

## Frontend

- React (Vite)
- Tailwind CSS v4
- React Router DOM
- Axios

## Backend

- Node.js
- Express.js
- MongoDB
- JWT Authentication
- bcryptjs
